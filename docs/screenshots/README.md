# Скриншоты

Сюда складываются скриншоты для основного README.

## Что нужно снять

- `chat-telegram.png` — пример диалога с ботом в Telegram, желательно
  с PNG-таблицей сравнения сортов в ответе.
- `webapp-chats.png` — список чатов в админке с маркерами `⛔ / ✓`.
- `webapp-chat.png` — лента сообщений конкретного чата.
- `webapp-whitelist.png` — страница `/whitelist` с тогглом режима
  и формой добавления.
- `architecture.png` — схема компонентов (Excalidraw / draw.io).

## Размер

Скриншоты Retina скейлить до **≤ 1600 px** по длинной стороне и
сохранять PNG (для UI) или JPEG q=85 (для фотографий). Не больше
~300 KB на файл, чтобы не раздувать репозиторий.
