# Слой 3 — Справочники для досье

Эти документы — **единый источник правды** для всех досье на сорта. Не повторять их содержимое в каждом досье — ссылаться через `[[ref:имя-файла]]`. Любая правка здесь автоматически распространяется на все ~600 сортов при следующем перегенерации.

## Принципы

- **Что здесь:** правила, формулы, классификации, типовые таблицы, контакты — то, что одинаково для любого сорта.
- **Что НЕ здесь:** факты о конкретных сортах (это в `data/dossiers/`), сырые данные карточек магазина (это в `data/catalog/`).
- **Источники:** для каждого числового утверждения — ссылка или диапазон из нескольких источников. См. блок Е2 и Е3 промпта.

## Используется так

```markdown
В разделе досье:

> Сорт Уралец с заявленной зимостойкостью −43 °C даёт для Томска запас прочности +2 °C
> по формуле из [[ref:hardiness_formula]]. По классификации [[ref:groups_b1]] это
> Группа 1 (штамб без укрытия) с оговорками. Подвой — см. [[ref:rootstocks]].
```

При генерации досье через агента: вставлять содержимое нужных файлов как **кешируемый** контекст (cache_control breakpoint после Слоя 3). Это даст ×10 экономии токенов на 28+ прогонах.
