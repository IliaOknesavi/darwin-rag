# Опыление в холодном цветении Томска (В5–В6)

## Цветение в Томске — общее

Цветение основных плодовых культур в Томской области приходится на период, **совпадающий с возможными возвратными заморозками** (см. [[ref:spring_frosts]]):

| Культура | Сроки цветения в Томске |
|---|---|
| Жимолость, смородина | Конец мая |
| Яблоня, груша | Первая–вторая декада июня |
| Слива, вишня | Первая декада июня |
| Облепиха | Начало июня |
| Земляника | Первая половина июня |
| Малина | Третья декада июня |

Сдвиг относительно Центрального региона — **+2–3 недели позже**. Использовать эту поправку в разделе 8 каждого досье.

## Активность насекомых-опылителей

| Опылитель | Активность с температуры | Активность с погоды |
|---|---|---|
| Медоносная пчела | +12…+14 °C | Безветрие, отсутствие дождя |
| Шмель | **+6…+8 °C** | Работает в моросящий дождь, ветер до 5 м/с |
| Осмии (одиночные пчёлы) | +10 °C | Работают в облачную погоду |
| Журчалки, мухи | +10 °C | Менее эффективны (короткий хоботок) |

**Главный риск для Томска:** холодная неделя в первой декаде июня (типичный сценарий) → пчёлы не летают → яблони цветут впустую. Меры подстраховки:

1. **Привлечение шмелей** — не косить одуванчики, клевер, мать-и-мачеху в радиусе 50 м от участка с яблонями. Сохранять кучи сухих листьев и мха зимой (места гнездования шмелиных маток).
2. **Установка шмелиных домиков и домиков для осмий.** Стандартный размер: 15×15×30 см, отверстие 18 мм для шмелей, 8 мм для осмий.
3. **Опрыскивание раствором мёда / сахара** во время цветения для дополнительного привлечения насекомых (1 ст. ложка мёда на 1 л воды).

## Расстояния между основным сортом и опылителем

| Культура | Расстояние |
|---|---|
| Яблоня | 25–50 м |
| Груша | 25–50 м |
| Слива | 30–40 м |
| Вишня | 30–40 м |
| Облепиха | 5–10 м (на ветре переносится дальше) |
| Жимолость | 1–2 м (через 1 куст) |
| Голубика | 1–2 м |

На небольшом участке (6–10 соток) при посадке деревьев на стандартной дистанции 4–5 м опыление происходит надёжно — все деревья оказываются в радиусе влияния.

## Самоплодность — ключевая характеристика

В досье обязательно указать тип:
- **Самоплодный** — даёт урожай без опылителя (но с ним урожайность выше).
- **Частично самоплодный** — даёт 20–40% урожая без опылителя.
- **Самобесплодный** — без опылителя плодов не будет.

Большая часть культурных сортов яблони и груши — **самобесплодны** или частично самоплодны. Без опылителя сада не получится. Об этом нужно говорить прямо в досье на каждый самобесплодный сорт.

## Особые случаи

### Двудомные культуры
**Облепиха, актинидия, лимонник китайский** — нужны мужские и женские особи отдельно. Соотношение **1 мужской на 5–8 женских**, мужские располагаются с наветренной стороны.

### Несовместимые сортогруппы

**Жимолость** — практически все сорта **самобесплодны**, нужны 2–4 разных сорта одного срока цветения. Жёсткого деления на «несовместимые группы» по типу яблони у жимолости **нет** — рекомендации идут по конкретным парам сорт–опылитель, проверенным в селекционной работе (Бакчарский опорный пункт НИИСС, питомник в Бакчаре).

Проверенные пары опылителей (по данным bakcharsad.ru и сортоиспытаний):
- **Бакчарский великан** — опыляется сортами Нимфа, Амфора, Сильгинка, Гордость Бакчара, Лазурная, Югана.
- **Сильгинка** — опылители: Бакчарский великан, Дочь великана, Восторг, Югана.
- **Югана** — опылители: Дочь великана, Стрежевчанка, Восторг, Бакчарский великан.
- Сорта **бакчарской селекции** (Бакчарский великан, Сильгинка, Югана, Дочь великана, Восторг) **хорошо опыляют друг друга** — деления на «несовместимые подгруппы» внутри них нет.
- **Берель** — отдельный сорт селекции НИИСС (Барнаул), гибрид жимолости алтайской × камчатской; «берельской группы» как термина в помологии не существует.

Правило для досье: указывать конкретные сорта-опылители из источника (bakcharsad.ru, niilisavenko.ru), а не «группу».

**Голубика** — сортогруппы по высоте и срокам:
- Низкорослые (*V. angustifolium*): Норт Блю, Норт Кантри, Норт Скай.
- Среднерослые (*V. corymbosum × angustifolium*): Дюк, Патриот, Блюгольд.
- Высокорослые (*V. corymbosum*): Блюкроп, Спартан, Ранкокас.
- В пределах сортогруппы опыляют надёжно. Между сортогруппами — частично.

### Триплоиды
Сорта с **триплоидным набором хромосом (2n=51)** — **плохие доноры пыльцы** (стерильная или малоактивная пыльца). Нужны 2 опылителя для основной яблони + второй диплоидный сорт для самого триплоида. Если основной сорт — триплоид, в досье это пометить.

**Классические триплоидные сорта:** Антоновка обыкновенная, Бельфлёр-китайка, Боровинка, Штрейфлинг (Осеннее полосатое), Суворовец, Мелба (по ряду источников).

**Триплоидные сорта селекции ВНИИСПК (Е. Н. Седов и др., от скрещивания 2x × 4x):** Августа, Александр Бойко, Бежин луг, Вавиловское, Дарёна, Жилинское, Масловское, Орловский партизан, Памяти Семакина, Праздничное, Рождественское, Спасское, Юбиляр, Яблочный спас и др.

**Внимание:** обычная **Папировка (= Белый налив, Алебастровое) — диплоид (2n=34)**, нормальный опылитель. Существует отдельная искусственная форма **Папировка тетраплоидная** (ВНИИСПК), которая используется только как донор диплоидных гамет в селекции триплоидов — её в продаже у обычных питомников нет. В досье массовых сортов Папировку как триплоид не помечать.

### Самостерильные пары
Некоторые пары сортов выглядят как опылители, но фактически **не опыляют друг друга**. Например: Мелба и Боровинка (по литературе — да; по практике сибирских питомников — слабо). Уточнять по первоисточнику конкретно для каждого сорта.

## Не-плодовые культуры

Для **декоративных, хвойных, травянистых** — описать способ размножения и нужду в партнёре для семян/ягод (например, рябина декоративная для ягод требует опылителя; туя самоопыляется). Если опылитель не нужен — пометить «не применимо» с объяснением.

## Не-растения

Для **биопрепаратов, коры лиственницы, удобрений, посадочных смесей** — пометить «секция опылителей не применима».
